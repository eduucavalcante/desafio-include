# Desafio Trainee Include Jr

Repositório destinado ao desafio final trainee da Include Jr, empresa júnior (EJ) de consultoria e desenvolvimento de software da Universidade Federal do Ceará, Campus Russas, que abrange os cursos de Engenharia de Software e Ciência da Computação.

O projeto consiste em um site institucional para a EJ de engenharia civil Vale J, bem como um painel administrativo que gerencia o conteúdo do site.

## Principais Tecnologias e Ferramentas Utilizadas

**Front-end:**

- Node.js
- TypeScript
- React
- Tailwind CSS
- Axios

**Back-end:**

- Python
- Django
- Django REST Framework
- MySQL
- JWT
- Cloudinary (para storage de imagens)
- Swagger (para documentação)
- Hospedagem:
    - Render (API)
    - Aiven (banco de dados)

**Ferramentas:**

- Trello (gestão ágil)
- Postman (testes de API)
- MySQL Workbench
- VSCode
